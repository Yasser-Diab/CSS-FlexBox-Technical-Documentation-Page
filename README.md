# CSS Flexbox Documentation Page

A responsive, mobile-first technical documentation page explaining the fundamentals of CSS Flexbox.  
Built using semantic HTML and modern CSS, with adaptive layouts for mobile, tablet, and desktop screens.

## Overview

This project presents a structured and readable guide to CSS Flexbox, covering:
- Core concepts
- Flex container vs flex items
- Main axis and cross axis
- Common Flexbox properties
- Practical code examples

The page is designed to be easy to navigate and optimized for learning, following a documentation-style layout.

---

## Features

- Fixed navigation bar with anchor links
- Smooth scrolling with scroll offset handling
- Mobile-first responsive design
- Adaptive layout for portrait and landscape screens
- Syntax-highlighted code blocks
- Clean typography and readable spacing
- Footer with author and external links

---

## Technologies Used

- HTML5 (semantic structure)
- CSS3
  - Flexbox
  - Media queries
  - Responsive units
- No JavaScript required

## Layout & Responsiveness

- **Mobile (default):**
  - Fixed top navigation
  - Scrollable navigation list
  - Single-column content layout

- **Tablet:**
  - Increased font sizes
  - Improved spacing and readability

- **Desktop / Landscape:**
  - Fixed side navigation
  - Content shifted for sidebar layout
  - Persistent footer alignment
